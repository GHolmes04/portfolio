Fantasy Football Auction Assistant

A Fantasy Football App to help keep track of your leagues auctions

New

As an active fantasy football player I have joined many auction style leagues. Some of the most fun is when everyone gets together in person to nominate bid and fill their team with players. This application allows users to keep track of the auction results

Currently the app's Back-End is build using Ruby on Rails and Postgres. While the responsive front end is built with HTML5, Custom BootStrap, Custom CSS, JavaScript, JQuery, and Handlebars. In addition this app is integrated with MyFantasyLeagues API to fetch NFL player information. MFL API

Eventually users of MyFantasyLeague.com will be able to auto update the rosters no their league websites with a push of a button.
